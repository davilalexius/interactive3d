Обязательный код 3D:
<div id="i3d">
<img id="my3d1" src="путь к первому файлу-3D/my3d_1.jpg">
</div>

в файле 3dhelper.interactive3d.js
demo=true; //код для демонстрационного режима. false чтобы работал на вашем сайте
кусок кода для настройки 3d
 $("#i3d").interactive_3d({
     frames: summ, //!!Настраивается скриптом! количество картинок в папке, чем больше, тем плавнее анимация.
     cursor: "move", // The CSS-стиль курсора. По-умолчанию "move"
     speed: 0, // Скорость вращения в мс. Если у вас мало кадров  и вращение слишком быстро и не плавное увеличьте до 50-100.
     entrance: true, // Входная анимация. Напишите false чтоб выключить. The default value is true.
     preloadImages: true, // Позволить загрузить все кадры до анимации.
     touchSupport: true, // Разрешить touch events для мобильных. Если противоречия с сайтом- выключайте.  По умолчанию true.
     loading: "Loading..", // Если preloadImages true. индикатор загрузки(поддерживает HTML).false чтоб выкл
     autoPlay: false //автопрокрутка, либо false, !!!либо скорость смены в миллисекундах
    });

В папке 3d-рисунка имена файлов должны иметь вид:my3d_N.jpg , где N= число, начиная с 1(!!!не с 0);
пример:
images/3d:
my3d_1.jpg
my3d_2.jpg
my3d_3.jpg

Если кроме основного 3d-рисунка на странице есть несколько других, 
которые так же имеют свою 3D-папку, используйте для них <img class="img3dclass">

Наличие jquery  обязательно!
Например:
<script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.9.0/jquery.min.js"></script>
